Detailed Description

Responsible for procedural generation of ground rows and map elements.

This class dynamically creates rows of ground nodes based on the camera's visible area. It ensures that the map continuously extends as rows move backward, creating an infinite scrolling environment.

Creates a new row of ground nodes.
