Detailed Description

Controls an active tower that fires bullets in a patterned sequence.

When the player presses the Q key, a coroutine starts that continuously spawns bullets in opposite directions, creating a sprinkler-like firing pattern.

Coroutine responsible for spawning bullets in a pattern.

Bullets are spawned in two opposite directions. Each shot changes the angle, creating a rotating sprinkler-like firing pattern.

Returns
IEnumerator Required for Unity coroutine execution.

Loop generating bullets until the maximum number of shots is reached.

Creates two bullets fired in opposite directions.

Calculates the firing angle for the bullet.

The angle depends on the current shot number and determines the direction in which the bullet will travel.

Sets the movement direction of the bullet.

Waits for the specified fireRate before spawning the next bullets.

Resets the coroutine reference after the firing sequence finishes.
